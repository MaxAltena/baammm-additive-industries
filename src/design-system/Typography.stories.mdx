# Typography

## Fonts

The font Frank New is used for headings, combined with Open Sans for content and longreads.
Titles have a font size of 30px, whereas subtitles have 20px and regular text is sized at 15px.

Headings stand out because of their font size and font weight, making it clear to the user that the text marks the start of a new section.
